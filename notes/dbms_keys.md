
Keys are used to uniquely identify records in a database table.
Types of Keys
1. Primary Key
A primary key uniquely identifies each record in a table.
Example:
Student_ID
2. Foreign Key
A foreign key is used to connect two tables.
3. Candidate Key
All the possible keys that can uniquely identify rows.
 4. Super Key
A set of one or more columns used to identify records.

 Importance of Keys
- Prevent duplicate data
- Maintain relationships between tables
- Improve data organization
